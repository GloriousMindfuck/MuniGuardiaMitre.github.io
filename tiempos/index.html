¿Cómo funciona el registro?

1. Completa el formulario: Ingresa tu nombre, selecciona el sendero y registra tu tiempo.

2. Procesamiento: Los datos se procesan automáticamente mediante Google Forms.

3. Actualización: El ranking se actualiza cada 15 minutos con los nuevos registros.

4. Privacidad: Solo tu nombre/apodo aparece públicamente, tus datos personales están protegidos.

Importante: Para que el sistema funcione correctamente, necesitas configurar tu propio Google Forms. Las instrucciones están en la documentación del proyecto.
